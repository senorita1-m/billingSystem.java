# Billing System (Java)

This is a simple **Billing System** built using Java.

## Features
- Enter item name
- Enter price and quantity
- Calculates:
  - Total amount
  - GST (5%)
  - Final payable amount

## How to Run (locally)
1. Save `Main.java` in a folder.
2. Open terminal/command prompt in that folder.
3. Compile:
   javac Main.java
4. Run:
   java Main
